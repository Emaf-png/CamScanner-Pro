## 🚀 المميزات

| الميزة | الوصف |
|--------|-------|
| 🔍 **بصمة ذكية** | اكتشاف الشركة المصنعة والطراز تلقائياً |
| 💣 **ثغرات حقيقية** | CVE-2017-7921/7925 (Hikvision) + CVE-2021-33044 (Dahua) |
| 📸 **التقاط RTSP** | تصوير إطار من البث المفتوح تلقائياً |
| 🌐 **بحث Shodan** | بحث عالمي عن الكاميرات المكشوفة |
| 📊 **تصدير JSON** | تقارير منظمة للتكامل مع الأدوات الأخرى |
| 🔐 **فحص كلمات المرور** | اختبار كلمات المرور الافتراضية والضعيفة |

## 📥 التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/Emaf-png/CamScanner-Pro.git
cd CamScanner-Pro

# صلاحية التنفيذ
chmod +x camsec_pro.sh

# تشغيل
./camsec_pro.sh

# أول مرة: اختر 6 لتثبيت المتطلبات
```

# الخيار
1 تدقيق كامل (اكتشاف + فحص)
2 فحص هدف واحد

5 اختبار كلمات المرور
6 تثبيت المتطلبات
7 إعداد Shodan
8 بحث Shodan
